WifiDirect + ProxyServer
========================

Share your internet connection.

Creates a WifiDirect group which any Wifi client can connect.
Also has a proxy server running to proxy web requests.

WifiDirect is a peer-to-peer connection that works besides your normal Wifi connection.
The WifiDirect owner uses IP 192.168.49.1 and clients recieve IPs in this range.

The WifiDirect server is restarted every 15 mins of idle activity to keep it alive.
